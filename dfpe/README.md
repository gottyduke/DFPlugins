## 编译

使用主`dfpe`项目的`!rebuild`脚本生成解决方案.  

### 需求
+ [CMake](https://cmake.org)
+ [vcpkg](https://github.com/microsoft/vcpkg/releases)
+ `/std:c++23` 或 `/std:latest`
